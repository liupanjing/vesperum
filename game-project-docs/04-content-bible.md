# 内容与世界观圣经

## 1. 世界观基调

这是一个魔王被打败后的世界。

初代勇者战胜魔王、推翻旧王朝后的第 80 年，世界进入共和国时代。新世界拥有民主议会、完善法律、魔导科技、公共教育、商业繁荣和魔王监测体系。

官方叙事认为：

- 魔王已死，但污染可能复活。
- 勇者是世界安全的最终保险。
- 民主、法律和监测体系让人类避免重蹈旧时代覆辙。

实际情况：

- 魔王监测体系检测到的不是纯粹邪恶，而是矛盾浓度。
- 魔王已经从怪物变成一种社会标签。
- 很多人希望勇者替自己消灭敌人。
- “魔王”可能是罪犯、异见者、群众、制度，也可能是勇者自己。

## 2. 主题边界

本作要批判社会现状，但不能写成单薄讽刺。

原则：

- 不把任何群体写成天然邪恶。
- 不把民主写成天然正确。
- 不把强力秩序写成天然错误。
- 不把穷人、富人、男人、女人、官员、商人、群众写成单一面孔。
- 每个群体都要有受害者、加害者、沉默者和获利者。

## 3. 主角

主角不是传承序列中的新勇者，而是共和国白塔体系中的白塔巡证官，同时是第十三号圣剑承载者。

初代勇者死后，世界再也没有公认的新勇者。共和国、教会和白塔都试图继承勇者遗产，但他们能复制的是制度、仪式、档案和训练，不能复制“勇者”本身。因此，主角的身份从一开始就是矛盾的：他拥有圣剑反应和合法授权，却不天然拥有勇者的道德合法性。

可自定义：

- 姓名。
- 性别。
- 外观。

固定背景：

- 出身圣遗院。
- 从小接受共和国、教会、白塔和议会共同培养。
- 相信自己被选中是为了调查魔王反应，并在必要时启动圣剑裁决。
- 一开始相信世界虽然有问题，但总体已比旧时代更好。

称呼规则：

- 官方文书称主角为“白塔巡证官”或“圣剑承载者”。
- 民间、媒体、受害者和反对者常称主角为“勇者”。
- 教会内部称主角为“承剑者”或“被验者”。
- 玩家是否接受“勇者”这个称呼，会影响台词、自我认知、结局标题和部分能力命名。

主角核心问题：

> 当所有人都要求勇者站在自己这边时，勇者到底代表谁？

成长方向：

- 前期：相信自己也许能成为人们期待的勇者，并寻找魔王。
- 中期：发现自己每次选择都会制造新的敌人。
- 后期：意识到魔王可能不是一个对象，而是一种命名敌人的机制。
- 终局：玩家决定主角要接受勇者之名、放弃勇者之名，还是成为救世主、裁决者、圣王、魔王。

## 4. 圣剑

圣剑是勇者特有的权柄，不是普通战斗武器。

普通冲突通过卡牌对战解决，圣剑不进入常规牌组，也不作为普攻或技能使用。它的意义在于：当勇者拔出圣剑时，可以直接杀死对方，或将某个对象从政治、法律和社会意义上彻底裁决。

因此，圣剑不是“更强的剑”，而是“越过过程的最终答案”。

圣剑使用场景：

- 战前直接处决目标，跳过卡牌对战。
- 战中强行终止战斗。
- 战后对被压制者执行处决。
- 剧情中对被判定为魔王反应源的对象进行最终裁决。

圣剑后果：

- 短期内最有效，能迅速终止危机。
- 武力值和恐惧勇者标签显著上升。
- 普通人可能因此获得安全感。
- 反对者会认为勇者越过法律和公共讨论。
- 队友会根据自身立场重新评价主角。
- 章节结算会出现“果断拯救”与“不可反抗”的双重记录。

圣剑也是主角状态的视觉化。

圣剑形态：

- 初始形态：白银圣剑，光芒柔和。
- 正义形态：剑身出现法纹，更具裁决感。
- 仁慈形态：剑光偏暖，象征保护和净化。
- 秩序形态：剑身像法槌或结界核心。
- 自由形态：剑形轻盈，象征挣脱束缚。
- 真实形态：剑光透明，象征破除幻象。
- 安稳形态：剑光厚重，象征屏障和压制。
- 高武力形态：剑光变冷、变重、变宽，像王权或支配。

注意：高武力形态不一定丑恶。它可以庄严、强大、令人安心，也令人不敢直视。

## 5. 魔王监测体系

监测体系由共和国、教会和魔导学会共同维护。

表面功能：

- 监测魔王污染。
- 预警灾难。
- 指引勇者行动。

实际功能：

- 监测矛盾浓度。
- 识别可能失控的社会冲突。
- 为共和国管理危机提供合法性。
- 在必要时制造共同敌人。

## 6. 核心组织

### 共和国议会

世界新秩序的象征。它确实比旧王朝更开放、更合法、更重视民生，但也会为了稳定隐藏真相。

### 教会

负责解释勇者、魔王和圣剑的神圣意义。内部有人真诚地守护人民，也有人维护宗教权威。

### 魔导学会

负责圣塔和监测技术。成员倾向相信数据，但他们的数据并不中立。

### 圣遗院

研究、保管并解释初代勇者遗产的机构。它培养档案官、巡证官、随行人员和圣塔记录员，但不宣称自己能培养真正的勇者。圣遗院教育主角相信使命，也在无形中塑造主角对世界的单纯理解。

## 7. 队友设定

### 法律骑士

定位：守序、防御、反击。  
价值观：法律虽不完美，但没有法律只会更糟。  
冲突：如果玩家长期绕过法律使用私刑，他会质疑甚至离队。  
可能结局：成为圣王路线的宰辅，也可能成为反抗武力勇者的最终骑士。

### 逃亡魔女

定位：幻术、调查、异常状态。  
价值观：被官方称为污染者的人，不一定真的邪恶。  
冲突：如果玩家过度相信圣塔和官方叙事，她会与玩家冲突。  
可能结局：帮助揭露魔王监测体系，也可能被玩家亲手封印。

### 商会佣兵

定位：爆发、道具、现实主义。  
价值观：利益比口号诚实，穷人也需要市场，理想不能当饭吃。  
冲突：如果玩家不断牺牲现实经济换道德胜利，他会反对。  
可能结局：成为玩家治理世界的经济执行者，或成为反圣王地下网络的一员。

### 无名修女

定位：治疗、净化、调停。  
价值观：宽恕不能替受害者做决定，复仇也不能自动治愈伤口。  
冲突：她会同时反对廉价仁慈和彻底断罪。  
可能结局：在真结局中帮助主角放弃绝对勇者身份。

## 8. 第一章：玻璃镇

主题：真相与安稳。

玻璃镇是共和国宣传中的模范城镇。这里零犯罪、街道整洁、居民礼貌、幸福指数极高。

秘密：

镇上的幸福依赖“裂纹者”制度。被认为会破坏幸福指数的人，会被剥夺身份、迁入地下区域。他们仍然劳动，却不再被统计为镇民。

主要势力：

- 镇议会：维护模范城镇。
- 裂纹者：被制度排除的人。
- 普通居民：享受安稳，也害怕真相。
- 共和国监察官：希望勇者维持官方叙事。

关键 NPC 示例：

### 镇长

标签：男 / 官员 / 富人 / 守序 / 既得利益者 / 父亲  
表面：温和、礼貌、讲公共利益。  
真实：知道制度残酷，但认为这是最低代价。  
可能 Boss：白玻璃骑士的召唤者。

### 裂纹者领袖

标签：女 / 穷人 / 流民 / 自由 / 被牺牲者 / 激进  
表面：坚决、愤怒、不信任勇者。  
真实：既想夺回身份，也准备牺牲地上居民换取关注。  
可能 Boss：裂纹者领袖。

### 普通面包师

标签：男 / 群众 / 平民 / 商人 / 守序 / 本地人  
表面：善良，愿意给主角免费面包。  
真实：知道有些人消失，但不想追问。  
用途：表现普通人对安稳的依赖。

### 地下农夫

标签：男 / 穷人 / 农民 / 父亲 / 被牺牲者 / 受害者家属  
表面：沉默，对勇者敬畏。  
真实：他的孩子因身份抹除无法治疗。  
用途：测试玩家是否理解隐藏账本。

## 9. 后续章节方向

### 黄金城

主题：唯利是图与合法剥削。  
核心问题：如果一切剥削都写进合法合同，勇者该如何挥剑？

### 镜湖邦

主题：男女对立与被制造的仇恨。  
核心问题：当双方都有伤口，也都有获利者时，勇者站哪边都会被另一边称为魔王。

### 无声修道院

主题：道德洁癖与舆论审判。  
核心问题：公开所有人的罪与羞耻，社会会更正义，还是更残酷？

### 王都共和国

主题：寻找魔王本身如何制造魔王。  
核心问题：世界是否需要一个共同敌人来维持自身统一？
