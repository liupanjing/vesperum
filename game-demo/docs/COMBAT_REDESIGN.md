# 战斗系统重设计记录

## 当前结论

上一版“三槽战斗：HP / Will / Fear”不采用。

新的方向是：

> 普通冲突使用卡牌对战，圣剑作为勇者特有的非战斗裁决能力。

也就是说，圣剑不进入牌组，不作为攻击牌或技能牌使用。它可以直接杀死目标、终止冲突或执行最终裁决，但会带来强烈的武力、恐惧和社会后果。

## 设计原则

新的战斗系统需要同时满足：

1. 有明确 RPG 对抗感。
2. 有卡牌构筑和对局乐趣。
3. 武力可以真的变强，并带来效率。
4. 使用武力和圣剑的后果不能只在结局才出现。
5. Boss 不一定是邪恶者。
6. 卡牌对战结果能反过来影响隐藏群众账本。
7. Demo 阶段能低成本实现。

## 圣剑定位

圣剑是勇者特权，不是普通武器。

可用方式：

- 战前拔剑：跳过卡牌对战，直接处决目标。
- 战中拔剑：强行终止对局。
- 战后拔剑：对被压制者执行最终裁决。
- 剧情拔剑：对被判定为魔王反应源的对象执行使命。

后果：

- 武力值上升。
- 恐惧勇者标签上升。
- 冲突短期快速结束。
- 支持强力秩序的人群评价上升。
- 反对越权裁决的人群评价下降。
- 队友可能质疑或支持。

## 卡牌对战方向

卡牌对战代表普通冲突中的策略、资源、证据、群众支持和个人能力。

牌组来源：

- 基础行动牌。
- 显性数值牌。
- 证据牌。
- 队友牌。
- 群众牌。
- 武力牌。

卡牌胜利默认代表压制、逼退、瓦解或击败，不默认杀死。

## 不要做的方向

- 不做复杂战棋。
- 不做大量装备词条。
- 不做三槽说服小游戏。
- 不把圣剑做成普通卡牌。
- 不做明显的和平/屠杀二分。

## 待定问题

- Demo 是否固定小牌组，还是允许轻度构筑？
- 圣剑是否在所有关键战斗中可用？
- 圣剑是否消耗资源，还是消耗世界对勇者的信任？
- 武力牌是否每次使用都写入隐藏账本？
- Boss 牌组需要做到多大差异才足够有辨识度？
